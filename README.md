# Chart Workshop

Welcome to the Chart Workshop, an interactive web application, which gives you the ability to create and customize various types of charts using Chart.js. This tool is designed to help you visualize data effectively by providing a user-friendly interface for chart creation. 

## Features
- **Chart Types**: Create different types of charts including bar and pie
- **Customizable Data**: Input your own data for chart generation
- **Dynamic Visualization**: Instantly see the results of your data inputs in the chart
- **Modal Display**: View charts in a modal window for better focus
- **Color Legend**: Understand chart colors with an accompanying legend
- **Responsive Design**: Works well on various screen sizes

## Getting Started
To get started with the Chart Workshop, follow these steps:
1. Click Log In button to access the workshop.
2. If you don't have an account, click Sign Up to create one.
3. Once logged in, navigate to the workshop page.
4. Input your desired chart parameters including type, title, and data.
5. Click the "Build!" button to generate your chart.
6. View your chart in the modal window and analyze the data visualization.
